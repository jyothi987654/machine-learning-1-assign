```bash
cd bert-tutorial
pip install torch transformers matplotlib numpy scikit-learn

```

For HuggingFace BERT: weights download automatically on first run (~440MB).

## What This Tutorial Covers

1. **The context problem**  why Word2Vec and GPT fail on ambiguous words
2. **Self-attention from scratch**  Q, K, V projections, scaled dot-product
3. **Multi-head attention**  12 parallel heads, concatenation and projection
4. **BERT encoder block** attention + residual + LayerNorm + FFN
5. **MLM pre-training** masked language modelling forces bidirectionality
6. **NSP pre-training** next sentence prediction for sentence-pair tasks
7. **Fine-tuning** classification, NER, QA, NLI with real accuracy results
8. **Layer freezing experiment** novel: how many layers to fine-tune?
9. **The BERT family** RoBERTa, DistilBERT, ALBERT, T5
